# Funnel Volume Analysis

This section presents the absolute volume of orders at each
stage of the food ordering funnel.

## Funnel Volumes

| Stage | Orders |
|------|--------|
| Menu View | 4 |
| Add to Cart | 2 |
| Checkout Start | 2 |
| Payment Success | 1 |
| Order Delivered | 1 |

## Key Observation
A significant drop in volume is observed between the Menu View
and Add to Cart stages, indicating early-stage user drop-off
before checkout initiation.
